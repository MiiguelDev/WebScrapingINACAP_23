Recuerden que deben crear un entorno virtual primero usando "python -m venv venv"
en caso de ya existir lo activas usando "venv/Scripts/activate".
No inclui la carpeta "venv" que guarda las dependencias porque en teoria deberia crearse con "python -m venv venv", si tiene algun 
incoveniente me avisan para agregarla.
luego abrir un terminal y ejecutar "pip install -r requirements.txt" en el directorio raiz del proyecto 
para instalar todas las dependencias, esto instalara dentro del proyecto las bibliotecas necesarias (ESTO DEBO REVISARLO OTRA VEZ, LAS DEPENDENCIAS NECESARIAS CAMBIARON)

Para ejecutar el proyecto debe ir a una terminal en el directorio raiz y ejecutar "python app.py" para ejecutar flask 
Dentro de la info que les muestra la terminal les dara una direccion ip que deben ejecutar haciendo Ctrl+click izquierdo o 
copiando y pegando la direccion. Cualquier cambio en el codigo se vera reflejado luego de refrescar la pagina (F5).

Para cerrar la app de flask solo deben cerrar la pesta√±a de la terminal, para salir del entorno virtual ejecuten "deactivate" o 
"venv\Scripts\deactivate" en caso de que no funcione, en la terminal deberia dejar de salir el sufijo (env) en la direccion
para comprobar que realmente salieron.
